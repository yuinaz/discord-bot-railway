SatpamBot – Installation Guide (Updated 2025-08-09 15:00:06 UTC)

1) Python Setup
- python -m venv .venv && source .venv/bin/activate  (Windows: .venv\Scripts\activate)
- pip install -r requirements.txt

2) Env Profiles
- Lokal: .env.local (ENV_PROFILE=local, DISCORD_TOKEN=..., dst)
- Render: set env di dashboard (opsional .env.prod sebagai fallback)
- Jalankan lokal: ENV_PROFILE=local python main.py

3) Dashboard
- http://localhost:8080/dashboard → banner ENV, Validate Env, stats, security editor

4) Monitoring
- /healthcheck (UptimeRobot), /heartbeat (status), /widget (embed), tools/windows_notifier.py (toast)

5) Desktop Widget (Electron)
- cd desktop-widget && npm install
- Set env: WIDGET_BASE, WIDGET_DASHBOARD, WIDGET_TOKEN, WIDGET_SECRET
- npm start (dev) / npm run build:win (installer .exe, auto-update via GitHub Releases)

6) Keamanan Endpoint
- POST /restart: header X-Widget-Token must equal DESKTOP_RESTART_TOKEN
- GET /desktop-status: HMAC headers X-Widget-Timestamp + X-Widget-Signature (DESKTOP_STATUS_SECRET)

Troubleshoot: cek DISCORD_TOKEN, intents, Tesseract, VT cache TTL, token/secret widget.
